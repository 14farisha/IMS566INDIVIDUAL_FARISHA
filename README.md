## Project title
Portal Pelajar

## Description
Portal Pelajar ialah aplikasi website yang lengkap dengan login, dashboard, courses, dan lecturers pages.

## Features
- Authentication dengan hardcoded credentials
- Sidebar navigation konsisten
- Dashboard dengan adanya carta bar mingguan kehadiran pelajar dan carta pai peratus subjek
- Courses dan Lecturers data tables
